    Cara Kerja Website

    • - HTML, CSS, Dan Javascript: Konten website biasanya ditulis dalam HTML (Hypertext Markup Language), gaya ditentukan oleh CSS (Cascading Style Sheets), dan interaktivitas ditambahkan melalui JavaScript.
    • - Browser: Ketika seseorang mengunjungi sebuah website, browser pada perangkat pengguna mengirimkan permintaan ke server untuk mendapatkan halaman web.
    • - Server: Server menerima permintaan tersebut dan melakukan pemrosesan server-side jika diperlukan (misalnya, membaca data dari database).
    • - Response: Server mengirimkan kembali halaman web yang diminta ke browser pengguna.
    • - Rendering: Browser mengambil respons tersebut dan merender halaman web, memperlihatkan konten dan tata letak yang diinginkan.

    Hubungan antara Web Server, Domain, dan Hosting:

  1. - Web Server: Ini adalah komputer atau perangkat keras khusus yang menyimpan dan menyajikan halaman web kepada pengguna yang memintanya. Web server mengelola permintaan dari browser pengguna dan mengirimkan kembali halaman web yang diminta. Contoh web server populer termasuk Apache, Nginx, dan Microsoft IIS.
  2. - Domain: Domain adalah nama yang diberikan ke web server untuk menunjukkan lokasi penyimpanan halaman web. Ini memungkinkan pengguna untuk mengakses halaman web melalui URL yang sesuai. Contoh domain populer termasuk .com, .org, dan .net.
  3. - Hosting: Hosting adalah proses penyimpanan dan pengelolaan halaman web yang dilakukan oleh web server dan domain. Ini memungkinkan pengguna untuk mengakses halaman web melalui URL yang sesuai.

    Hubungan Ketiganya:

  1. Saat seseorang mengunjungi suatu website, browser akan menggunakan alamat domain untuk mencari tahu alamat IP server yang sesuai dengan domain tersebut melalui DNS.
  2. Setelah alamat IP ditemukan, browser mengirimkan permintaan ke server menggunakan alamat IP tersebut.
  3. Server, yang dihosting pada suatu tempat, menerima permintaan tersebut dan mengirimkan halaman web yang diminta kembali ke browser melalui jaringan.
  4. Browser merender halaman web dan menampilkannya kepada pengguna.
